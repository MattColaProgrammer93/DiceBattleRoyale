# DiceBattleRoyale
The game of dice battle royale where a certain number of players can fight each other by using dice to roll for a number. Items can also influence the game by increasing a player's basic stats.

To run the game, make sure that all java files are within  in the project's src. If files aren't in the src, the game will not start until all files are present in the src. If you can also take the project and readme file out of the DiceBattleRoyale folder if you don't use github for eclipse.
